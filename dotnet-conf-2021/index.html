About The Conference

.NET Conf 2021 is a free virtual developer conference co-organized by the Bangalore DotNet UG (BDotNet), Dot Net Learners House community and Microsoft. Come and Join us to learn about the latest and greatest of Microsoft Technologies directly from Microsoft folks, Microsoft MVPs, Industry experts, and Community rockstars.

    Blazor Azure B2C Authentication and Authorization

    By Jagadesh Julapalli
    Azure B2C is an identity and access management solution for consumer-facing web and mobile applications. Customers use their preferred social, enterprise, or local account identities to get single sign-on access to your applications and APIs. In this session, we will discuss the importance of Authentication and Authorization and securing the Blazor app with Azure AD B2C.

    Let's build Minimal APIs using .NET 6

    By Suhas Parameshwara

MVC is a comprehensive framework that offers many great features out-of-the-box. In this session we'll explore A. what are Minimal API's, B. Pros & Cons of Minimal API's C. How can we build Minimal API's using .NET 6

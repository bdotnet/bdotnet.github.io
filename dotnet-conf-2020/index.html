About The Conference

.NET Conf 2020 is a free virtual developer conference co-organized by the Bangalore DotNet UG (BDotNet), XHackers UG community and Microsoft. Come and Join us to learn about the latest and greatest of Microsoft Technologies directly from Microsoft folks, Microsoft MVPs, Industry experts, and Community rockstars.

    What's new with Blazor ?

    By Swaminathan Vetri

From improved APIs for working with the browser to CSS isolation to a variety of performance improvements, you’ll leave this presentation with rundown of everything you’ll be able to do in the latest version of Blazor.

  • 12:30 PM - 01:15 PM IST

    The Why and How of C# 9.0

    By Igor Rončević
    In this talk, we start with Why. After explaining the motivation behind individual C# 9.0 features, we demonstrate the How through live-refactoring of an existing C# 8.0 Blazor application to C# 9.0. By the time the talk and the refactoring end, you will: - understand the motivation behind the biggest C# 9.0 features like records and enhanced pattern matching. - learn which tools and techniques can help you efficiently refactor your existing code to C# 9.0. - understand how to safely use C# 9.0 on older, officially not supported, .NET frameworks and runtimes. - discover some smaller, less discussed, but very useful C# 9.0 features. Last but not least, we will have fun together, going through some C# memes and weird challenges that C# 9.0 made possible, like writing non-trivial programs without using any curly braces :-) ;-)
